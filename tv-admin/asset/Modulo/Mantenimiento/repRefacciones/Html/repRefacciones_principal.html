Zona de Mantenimiento

¿Para qué sirve esto?

Este módulo te permite "apagar" temporalmente el acceso para Admin. Al activar el modo mantenimiento, todos los usuarios serán desconectados y bloqueados a excepción de tu cuenta "Root". Úsalo únicamente cuando necesites actualizar código o hacer reparaciones críticas en la base de datos.
